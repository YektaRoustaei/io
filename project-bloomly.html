Bloomly Coaching Dashboard

React + TypeScript dashboard where I owned the architecture, integrations, and rollout. The CRM now handles 4x more sessions per coach with zero manual exports.

What I built

  • Implemented the dashboard shell in React + TypeScript with SSR, theme tokens, and full keyboard navigation.
  • Built WebSocket ingestion services and D3 charts so coaches see session KPIs within 500ms of change.
  • Shipped CSV import/export tooling that reduced onboarding time per studio from days to under an hour.
  • Documented components and analytics contracts in Storybook + Notion to unblock the internal dev team.
