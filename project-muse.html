ML nutrition

Nutrient ML Study

Prototype that predicts likely nutrient deficiencies and returns personalized recommendations using the NHANES 2013 to 2014 dataset. The work tied model training, API delivery, and application logic into one end-to-end software flow.

Modeling XGBoost, SHAP, feature analysis
Backend FastAPI with documented endpoints
Application Next.js delivery layer for predictions and recommendations

What I built

  • Trained models to predict nutrient deficiencies and identify multiple health risks from nutrition data.
  • Built a FastAPI service that exposed risk scores, top contributing features, and recommendation outputs.
  • Stored SHAP explanations and exported analysis visuals for review and interpretation.
  • Developed a Next.js application layer that translated raw model output into clearer recommendations and risk summaries.
  • Connected the application to the backend through a dedicated service layer for profile inputs and predictions.
