ML nutrition

Nutrient ML Study

Prototype that predicts likely nutrient deficiencies and returns personalized recommendations using the NHANES 2013 to 2014 dataset. The work tied model training, API delivery, and an explanatory frontend into one product flow.

Modeling XGBoost, SHAP, feature analysis
Backend FastAPI with documented endpoints
Frontend Next.js interface for risk and recommendation views

What I built

  • Trained models to predict nutrient deficiencies and identify multiple health risks from nutrition data.
  • Built a FastAPI service that exposed risk scores, top contributing features, and recommendation outputs.
  • Stored SHAP explanations and exported analysis visuals for review and interpretation.
  • Developed a Next.js frontend that translated raw model output into clearer product language.
  • Connected the UI to the backend through a dedicated service layer for profile inputs and predictions.
