Muse Habit App

React Native habit companion where I owned the codebase structure, offline data model, and release pipeline. Target was sub-2s cold starts on devices as old as iPhone 8.

What I built

  • Designed a component kit in React Native + TypeScript with strict prop typing so feature squads stopped duplicating UI.
  • Built offline-first streak/reminder logic with SQLite caching and delta sync via REST, keeping data drift under 0.5%.
  • Instrumented performance traces and trimmed bundle weight by 35%, cutting cold start to 1.6s on low-end hardware.
  • Set up release channels with CodePush + Detox tests so non-breaking UI tweaks could ship daily.
