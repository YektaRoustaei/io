Ori Studio UI Kit

Design-system engagement where I owned token architecture, React components, and regression tooling for the Ori team.

What I built

  • Defined design tokens (color, typography, spacing) in CSS variables with build scripts for React + marketing sites.
  • Built React components for cards, gallery grids, and booking forms, versioned in Storybook packages.
  • Set up Chromatic visual diffing and Percy snapshots, eliminating UI regressions after handoffs.
  • Delivered upgrade playbooks and CLI scaffolds so internal devs could adopt new components in minutes.
